Add Bowling Calculate in your website with shortcode

== Description ==

This plugin helps you to add bowling calculator in your website with shortcode.

**How to use?**

Put this shortcode `[bowling-calculator]` in your page/post where you want to place the calculator.

Thanks,
[Shahriar Ahmed Shovon](https://shahriarahmed.me)

== Installation ==

1. Unzip the downloaded 'bowling_calculator' zip file and upload the plugin folder into the wp-content/plugins/ directory. Alternatively, upload from Plugins >> Add New >> Upload Plugin.
2. Activate the plugin from the Plugins page.
